# PeakFit Website

A static fitness landing page with modern branding.

## Files
- `index.html` — main page
- `styles.css` — visual design and layout
- `script.js` — mobile menu behavior
- `logo.svg` — brand logo asset

## How to open
1. Open the folder `Desktop/fitness-website` in VS Code.
2. Open `index.html`, `styles.css`, `script.js`, or `logo.svg` from the Explorer.

## Preview in browser
- Open `index.html` directly in your browser.
- Or install the **Live Server** extension in VS Code and click `Go Live`.

## Notes
- Edit `styles.css` to change colors and layout.
- Edit `script.js` to update interactions.
- `logo.svg` is a scalable vector logo you can use anywhere.
